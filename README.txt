3 simples pasos para poder deployar el site en un server local

-gem install jekyll
-cd pigmalionstudios_blog
-jekyll serve

Abrir http://127.0.0.1:4000/ en el navegador

Luego, para crear un post, simplemente crear un archivo (o copiar el ya existente y cambiarle el nombre), siguiendo esta nomenclatura:
yyyy-mm-dd-sometitlewithoutspaces.markdown

Darle refresh, y el nuevo post estara disponible a traves de una entrada en la home (http://127.0.0.1:4000/)
